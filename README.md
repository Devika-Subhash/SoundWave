# SoundWave Music Player

A Spotify-inspired music player built using HTML, CSS, and JavaScript.

## Features

- Play/Pause songs
- Next/Previous controls
- Progress bar
- Volume control
- Search songs
- Active song highlighting
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML Audio API

## Future Improvements

- React Version
- User Playlists
- Favorites
- Fullscreen Player
